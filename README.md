# Product Quality & Decision Platform

Full-stack platform for managing product requirements, testing processes, and release risk assessment.

## Features

- Business requirements tracking (user stories, acceptance criteria)
- AI-powered test scenario generation
- Code change monitoring and impact analysis
- Test result tracking and analysis
- Release risk assessment and recommendations

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open `http://localhost:3000` in your browser

## Technology Stack

- Backend: Node.js, Express, SQLite
- Frontend: Vanilla JavaScript, HTML, CSS
